## 代码格式规范

全书的代码一定要统一格式，整个项目保持相同风格，在保持代码内容相同的情况下尽量追求简洁、美观

1. ### 空格和换行

空格和换行非常重要，在合适的地方空格会使代码的整洁度提升非常大，不至于全都挤在一起，下面的规范是咱们要用的，有些可能和余老师演示不一样，但契合大众，更加规范

- 函数体先换行再{}，if，for，while等工作流要先空格，{}，再换行

```C
 #include <stdio.h>
 int main()
 {                                   //函数的函数体{}要换行再打{，再回车回来
     int a[5] = {5, 2, 8, 1, 9};
     int i;

     for(i = 1; i < 5; i++) {        //for循环写完条件后空格，打{，再回车
         printf("%d\n", a[i]);
     }

     return 0;
 }
```

- 定义，描述语句的逗号要在后面加空格

```
int a, b, c;
 #include <stdio.h>                      //还是刚才的例子
 int main()
 {
     int a[5] = {5, 2, 8, 1, 9};         //逗号后面加空格，把每个元素分开，别堆在一起
     int i;

     for(i = 1; i < 5; i++) {            //虽然不是逗号，但这种描述性语句的分号;也要加空格
         printf("%d\n", a[i]);           //逗号后空格
     }

     return 0;
 }
```

- 运算符号，逻辑符号（包括 && 和 || ）两遍都要分开

```
int sum = a + b * c;
 #include <stdio.h>
 int main()
 {
     int a[5] = {5, 2, 8, 1, 9};         //等号两边空格
     int i;

     for(i = 1; i < 5; i++) {            //等于号，大于小于号两边空格，加减乘除两边都要加空格
         printf("%d\n", a[i]);
     }

     return 0;
 }
```

- for, if, while等工作流结构和上下分开一个换行

```C
 #include <stdio.h>
 int main()
 {
     int a[5] = {5, 2, 8, 1, 9};
     int i;
                                         //for循环上面有空换行
     for(i = 1; i < 5; i++) {
         printf("%d\n", a[i]);
     }
                                         //for循环下面有空换行
     return 0;
 }
```

1. ### 命名规范

- 定义函数的形参命名元素前要加_(下划线)

```
void fun(int _x， int _b[])
```

- 常亮命名要大写

```C
#include <stdio.h>

int A;                                    //常量大写

int main() 
{
    return 0;
}
```
